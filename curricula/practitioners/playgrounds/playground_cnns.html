Real-World CNN Applications
CNNs power most modern computer vision. Here's how they work in practice.

👤 Face Detection

CNNs detect faces by learning hierarchical features: edges in early layers, then eyes/nose/mouth shapes, then full face patterns.

🚗 Object Detection

Self-driving cars use CNNs to detect pedestrians, vehicles, signs. The network outputs bounding boxes and class labels.

The Key Insight: Hierarchical Features

CNNs don't jump from pixels to "face". They build up: edges combine into curves, curves into shapes (eye, nose), shapes into parts (face region), parts into objects. Each layer builds on the previous one. This hierarchy emerges automatically from training, not from manual design.

Why does spatial structure matter?
A dense network treats each pixel independently. A CNN exploits the fact that nearby pixels are related.

The Key Insight

Both images have exactly the same pixels and the same pixel values. But the scrambled version loses all spatial relationships. The "7" shape is destroyed. A CNN relies on detecting local patterns (edges, curves) that only make sense when pixels stay in their original positions. This is why CNNs work for images but not for tabular data where column order is arbitrary.
