📖 How to Read This Show ▼
📍 Scatter: Points stacked by class. X-axis is "risk score" the model learns from.
📈 Sigmoid: The learned curve. Converts score to probability. Vertical line = your threshold.
🟩 True Pos/Neg: Correct predictions. We want these HIGH.
🟨 False Pos: Predicted churn but didn't. Wasted retention offer ($FP cost).
🟥 False Neg: Missed a churner. Lost customer ($FN cost).
📉 PR Curve: Shows precision/recall tradeoff across all thresholds. Current threshold = orange dot.
📊 Calibration: Are predicted probabilities accurate? Points on diagonal = well-calibrated.
💰 Cost-Optimal: Threshold that minimizes FP×cost + FN×cost. Adjust sliders to see it change!
